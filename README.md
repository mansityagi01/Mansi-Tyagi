# 🌟 Mansi Tyagi - Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and achievements as a Computer Science & Engineering student specializing in Machine Learning and Full Stack Development.

## 🚀 Live Demo

**[View Live Website](https://mansi-tyagi.onrender.com)**

## ✨ Features

- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Interactive Animations** - Smooth scroll animations and premium glow effects
- **Dark Theme** - Elegant black theme with white/gray accents
- **Project Timeline** - Showcase of key projects with alternating layout and images
- **Certification Gallery** - Display of professional certifications with verification links
- **Skills Overview** - Comprehensive display of technical skills and technologies
- **Contact Integration** - Direct links to LinkedIn, GitHub, LeetCode, and email

## 🛠️ Tech Stack

### Frontend
- **React 18.3.1** - Modern React with functional components and hooks
- **TypeScript** - Type-safe JavaScript for robust code
- **Vite 5.4.2** - Lightning-fast build tool and dev server
- **Tailwind CSS 3.4.1** - Utility-first CSS framework

### Libraries & Tools
- **Lucide React** - Beautiful, consistent icons
- **Lottie React** - Smooth JSON-based animations
- **Three.js** - 3D graphics library (for future enhancements)
- **ESLint** - Code quality and consistency

## 🚦 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mansityagi01/portfolio-website.git
   cd portfolio-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

## 📦 Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist/` directory.

## 🎨 Customization

### Color Scheme
The website uses a pure black theme with gradient accents. To customize colors, edit the Tailwind classes in the components.

### Content
- Update personal information in `src/components/Hero.tsx`
- Modify project details in `src/components/Projects.tsx`
- Add/remove skills in `src/components/Skills.tsx`
- Update certifications in `src/components/Certifications.tsx`

## 🌐 Deployment

The website is deployed on **Render** with the following configuration:

- **Build Command:** `npm install && npm run build`
- **Start Command:** `npm start`
- **Environment:** Node.js

You can deploy your own version on:
- Render
- Vercel
- Netlify
- GitHub Pages
